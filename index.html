Formularios HTML

  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

    2. Serve para dar uma dica ao usuário, de um texto ou um caractere ao prencher um formulário.


  2. Na tag <input> com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

    3. Define o número de opções que o usuário deve selecionar.


  3. Na tag <input> para que serve o atributo "value"?

    4. Define o valor de entrada correspondente para um certo campo.


  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. O atributo: <input type="radio">


  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

    6. É um rótulo que define uma legenda para elementos do formulário. A tag "for" é um atributo de ligação entre o <label> e o <input>


  6. Na tag <option> para que serve o atributo "selected"?

    7. Serve para atribuir opções pré-selecionadas.


  7. Nas tags de formulário para que serve o atributo "required"?

    8. Serve para dizer ao usuário que é obrigatótio o preenchimento do formulário antes do envio.


  8. Na tag <form> para que serve o atributo "action"?

    9. Para indicar o endereço do qual o formulário será enviado.


  9. Na tag <form> para que serve o atributo "method"? Quais são os valores possíveis desse atributo?

    10. Define o endereço HTTP com que o formulário irá lidar com os dados recebido. Seus valores são: GET (Este método é útil para envios de formulários em que um usuário deseja marcar o resultado) e o método POST(Este método envia os dados do formulário como uma transação de postagem HTTP).


  10. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

    11. Atributo POST, porque ele tende a ser mais seguro e robusto que o método GET.


  11. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

    12. POST, por ele ser mais seguro na troca de dados.


  12. O que as tags <fieldset> e <legend> produzem?

    13. <fieldset> serve para um agrupamento de dados de um formulário,e a <legend> define uma legenda para o formulário.
